1- ¿Qué solución adoptarías para frente a un sitio web mal planificado?
	a) Replicar las máquinas servidoras.
	b) Reestructurar todo el sistema.
	c) Configurar varias máquinas, dedicando una a cada servicio que se ofrezca.
	d) Parchear el sistema.
Respuesta correcta: b

2- Para desplegar un servidor web nos centramos prioritarimente en:
	a) El trabajo inicial de configuración y la escalabilidad del sistema.
	b) El trabajo de mantenimiento, que el sistema sea escalable y con disponibilidad.
	c) La escalabilidad del sistema.
	d) La disponibilidad del sistema.
Respuesta correcta: b
